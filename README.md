# Kodiak Den

**Your private corner of the internet.**

Kodiak Den is a privacy-first social media project focused on user control, clean community features, and no creepy tracking.

## Brand source of truth

The official logo committed first for this project lives here:

```txt
assets/kodiak-den-logo.png
```

The web app imports that PNG directly. Do not replace it with SVG or a recreated mark unless we explicitly decide to redesign the brand system later.

## MVP: The Trail

Initial features planned:

- Profiles
- Roars/posts
- Comments
- Pawprints/reactions
- Pack/follows
- Post visibility controls
- Blocking/reporting
- Account export/delete
- Chronological feed
